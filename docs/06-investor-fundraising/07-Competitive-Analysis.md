## 4. Competitive Moats

1. **Privacy architecture** -- Self-hosted LLM is a fundamental architectural choice. Retrofitting cloud-first products for on-prem is extremely difficult.
2. **Multi-module data access** -- Our ERP ecosystem provides pre-built access to CRM, HRM, Finance, and 20+ other modules. Standalone assistants must build each integration from scratch.
3. **Agent platform** -- Our extensible agent framework creates a platform with community-contributed tools and agents, compounding value.
4. **RAG pipeline quality** -- Our semantic chunking + hybrid search + reranking pipeline consistently outperforms simple vector search approaches.
